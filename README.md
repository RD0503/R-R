# Victoria Rodríguez — Sitio Web Profesional

> Landing page de presentación para estudio jurídico independiente, orientada a captar clientes en el noroeste argentino y Buenos Aires.

---

## Descripción

Sitio web de una sola página (*single-page application*) diseñado como carta de presentación digital para la Dra. Victoria Rodríguez, abogada especializada en **Derecho Civil y Comercial** con ejercicio activo en **Jujuy, Tucumán y Buenos Aires**.

El objetivo del sitio es transmitir confianza, trayectoria y accesibilidad: que un potencial cliente pueda conocer a la profesional, entender en qué puede ayudarle y ponerse en contacto en el menor tiempo posible.

---

## Secciones del sitio

| Sección | Descripción |
|---|---|
| **Inicio** | Hero con nombre, título profesional y jurisdicciones de ejercicio |
| **Presentación** | Biografía, formación académica y métricas de trayectoria |
| **Áreas de Práctica** | Derecho Civil, Comercial, Laboral, Contratos, Asesoramiento Empresarial y Consultoría Jurídica |

| **Noticias y Logros** | Espacio para publicar reconocimientos y casos destacados |

---

## Tecnologías utilizadas

- **HTML5** semántico
- **CSS3** personalizado (`landing.css`) con variables, animaciones y diseño responsivo
- **Tailwind CSS** v3.4.4 (via CDN) para utilidades de grilla y espaciado
- **Google Fonts** — Playfair Display (títulos) + Inter (cuerpo)
- **Sin frameworks ni dependencias de JavaScript** — navegación y menú móvil resueltos con CSS puro

---

## Cómo usar

El sitio es completamente estático. No requiere servidor, instalación ni build.

```bash
# Clonar o descomprimir el proyecto
# Abrir directamente en el navegador
open index.html
```

Para publicarlo en producción basta con subir la carpeta completa a cualquier servicio de hosting estático: **GitHub Pages**, **Netlify**, **Vercel** o similar.

>  Mantener la carpeta `Imagenes/` junto a `index.html` en el mismo directorio raíz.
